よくあるご質問

アカウントを削除したいのですが？
アプリ内の「設定」 →「アカウント削除とデータ消去」から削除できます。 アプリを既にアンインストールしてしまった場合は、こちらのページからお手続きください。
歌会に参加するにはどうすれば？
招待コード（6文字）を知人からお聞きし、アプリ右上の「＋」→「招待コードで参加」から入会いただけます。 どなたでも参加できる「公開歌会」は、歌会タブ右上のコンパスのアイコンから探せます。
投稿した歌を削除したい
歌の詳細画面の右上にあるメニュー（…）から「削除」を選んでください。削除した歌は元に戻せません。
歌を画像として保存・共有したい
歌の詳細画面で、縦書きで表示されている歌そのものをタップするとプレビュー画面に進みます。そこから端末のギャラリーに保存できます。 解題済みの歌は作者名入り、そうでない歌は「よみ人しらず」として出力されます。
ふさわしくない投稿・利用者を見つけたら？
歌や評の詳細画面の三点メニューから、全ユーザーが通報またはブロックが行えます。通報を受けた投稿は歌会の主宰が24時間以内に確認し、規約違反と判断された場合は削除や投稿者のアカウント停止などの措置を講じます。 深刻な場合は上記のお問い合わせメールまでご一報ください。
通知が来なくて困っています
アプリ内の「設定」→ 通知項目、および各歌会の設定画面で、通知のオン/オフを切り替えられます。 公開歌会は初期状態で通知オフになっています。
