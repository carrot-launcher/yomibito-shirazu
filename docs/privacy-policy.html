プライバシーポリシー

最終更新日: 2026年4月24日

「よみ人しらず」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、日本の個人情報の保護に関する法律（以下「個人情報保護法」）その他の関係法令を遵守して個人情報を適切に取り扱います。本ポリシーでは、本アプリが収集・利用する情報、利用目的、ユーザーの権利について説明します。

事業者情報

  • 事業者名: Tenderloin Games
  • 個人情報に関するお問い合わせ窓口: tenderloin.games@gmail.com
  • 所在地その他の詳細は、上記窓口までお問い合わせください。

収集する情報

アカウント情報

本アプリは Google アカウントまたは Apple ID（Sign in with Apple）のいずれかによるログインを使用します。ログイン時に以下の情報が Firebase Authentication に保持されます。

  • Google または Apple が発行する一意の識別子（UID）
  • メールアドレス（Sign in with Apple で「メールを非公開」を選択した場合は、Apple が中継する非公開のリレーアドレスが保持されます）

これらの情報はアカウント管理および認証のためにのみ使用され、他のユーザーには公開されません。メールアドレスはアプリのデータベースには保存しませんが、Firebase Authentication サービスによって管理されます。アカウント削除時には Firebase Authentication の情報も完全に削除されます。

ユーザーが作成するデータ

  • 短歌の投稿内容（2〜50文字）
  • 評（コメント）の内容（1〜500文字）
  • リアクション
  • 歌会の名前、招待コード
  • 公開歌会の場合は趣意書（10〜200文字、参加していないログインユーザーにも公開されます）
  • 歌会ごとの表示名（匿名のペンネーム）
  • 歌人ID（6桁の数字コード）
  • 栞（ブックマーク）
  • 解題情報（ユーザーが「解題」機能で自らの作者名を公開した場合に、歌に紐づけて保存されます）
  • ブロックした歌人のリスト（あなたのアカウントに紐付けて保存され、他のユーザーには公開されません）

投稿内容に関する注意: 短歌の本文、評、歌会名、趣意書、歌会ごとの表示名などの「ユーザーが作成するデータ」は、同じ歌会の参加者（公開歌会の場合はさらに広い範囲のユーザー）に閲覧されます。本アプリはこれらの内容を秘匿情報として扱いません。氏名・住所・電話番号・勤務先・口座情報などの個人情報、業務上の機密情報、またはこれらに類するセンシティブな情報は投稿しないようご注意ください。投稿可能な内容については利用規約の禁止事項もあわせてご確認ください。

自動的に収集される情報

  • プッシュ通知用のデバイストークン: 通知機能のために使用します。
  • 利用状況データ（Google Analytics for Firebase）: アクティブユーザー数、セッション数、画面遷移などの匿名の利用統計を収集します。広告IDおよびSSAIDの収集は無効化しており、個人を特定する情報は収集しません。
  • クラッシュレポート（Firebase Crashlytics）: アプリの異常終了やエラーに関する情報（スタックトレース、デバイスの種類、OSバージョンなど）を自動的に収集します。これらの情報はアプリの品質改善のためにのみ使用します。

収集しない情報

本アプリは以下の情報を収集しません。

  • 位置情報
  • 連絡先
  • 写真・動画（端末のフォトライブラリから読み取ることはありません）
  • 広告ID（AAID / IDFA）
  • 財務情報

端末側の機能の利用

本アプリは、ユーザー操作に応じて以下の端末機能を利用します。いずれも端末内部で完結し、サーバーへ送信・保存されることはありません。

  • フォトライブラリへの書き込み: 「歌の画像化」機能で生成した画像を、ユーザーの明示的な操作により端末のフォトライブラリへ保存するために使用します。既存の写真・動画を読み取ることはありません。
  • プッシュ通知: 新しい歌・評・リアクション・裁きなどの通知を受け取るために使用します。端末の設定でいつでも無効化できます。

情報の利用目的

収集した情報は以下の目的で利用します。

  • アプリの機能提供（投稿、評、リアクション、通知など）
  • アプリの改善および利用状況の分析
  • クラッシュや不具合の検出と修正
  • プッシュ通知の送信

広告の表示や広告のパーソナライズには一切使用しません。

第三者への提供

ユーザーの個人情報を第三者に販売・貸与することはありません。ただし、以下のサービスを利用しており、これらのサービスのプライバシーポリシーが適用されます。

  • Firebase（Google LLC）: データの保存、認証、通知送信に使用
    Firebase プライバシーポリシー
  • Google Analytics for Firebase: 匿名の利用統計の収集に使用（広告ID収集は無効化済み）
    Google プライバシーポリシー
  • Firebase Crashlytics: クラッシュレポートの収集に使用
    Firebase Crashlytics のデータ収集について
  • OpenAI Moderation API（OpenAI, L.L.C.）: 投稿された歌および評の自動モデレーション（不適切な内容の事前フィルタリング）に使用。送信するのは投稿本文のみで、ユーザー識別情報（アカウントIDや表示名など）は含みません。OpenAI 社は Moderation API に送信されたデータを学習目的には使用しないことを明示しています
    OpenAI 利用規約 / OpenAI プライバシーポリシー

個人データの外国への移転

本アプリが利用する Firebase は、Google LLC（アメリカ合衆国）により運営されており、ユーザーのデータは米国を含む Google のデータセンター（複数の国・地域にまたがる可能性があります）に保管・処理されます。

Google LLC は、同社のプライバシーポリシーおよびFirebase のデータ処理方針に基づき、データの適切な保護措置を講じています。

また、投稿本文の自動モデレーション目的で、OpenAI, L.L.C.（アメリカ合衆国）が提供する Moderation API に投稿本文を送信する場合があります。送信されるのは本文のみで、ユーザー識別情報は含まれません。

個人データの安全管理措置

運営者は、ユーザーから取得した個人情報の漏えい、滅失またはき損の防止その他の安全管理のために、必要かつ適切な措置を講じます。具体的には、アクセス権限の制限、通信の暗号化（HTTPS）、Firestore セキュリティルールによるデータアクセス制御などを行っています。

漏洩等発生時の対応

万一、ユーザーの個人情報の漏えい、滅失、毀損その他の安全確保に係る事態が発生した場合、運営者は個人情報保護法その他の関係法令に基づき、個人情報保護委員会への報告および本人への通知を行うなど、適切な対応を速やかに実施します。

公開歌会における情報の公開範囲

公開歌会に所属している場合、以下の情報は当該歌会に参加していないログインユーザーにも閲覧される可能性があります。

  • 歌会の名前
  • 歌会の趣意書
  • 歌会のメンバー数・累計投稿数
  • 最新の歌（最新10首まで。ただし歌本体の内容のみ。詠んだ人の情報は含みません）

投稿した歌会が非公開歌会である場合、その歌の内容がメンバー以外に公開されることはありません。

データの保管期間

  • ユーザーが作成したデータ（短歌、評、リアクション、歌会情報、栞、通知履歴など）は、ユーザーがアカウントを削除するか、ユーザー自身あるいは歌会のオーナー等が当該データを削除するまで保管されます。
  • アカウント情報（Firebase Authentication 上の UID・メールアドレス）は、アカウントが削除されるまで保管されます。
  • 匿名化された利用統計データ（Google Analytics for Firebase）は、Google の定める最大14ヶ月間保持されます。
  • クラッシュレポート（Firebase Crashlytics）は、Google の定める保管方針に従って保持されます。

データの削除

ユーザーはアプリ内の設定画面にある「アカウント削除とデータ消去」機能、または専用のウェブページから、いつでも自分のアカウントと関連するすべてのデータを削除できます。削除されるデータには以下が含まれます。

  • 投稿した短歌
  • 書いた評
  • リアクション
  • 栞
  • 通知履歴
  • 所有する歌会（解散されます）
  • アカウント情報

削除を実行すると、上記のデータは即座にサーバーから削除され、復元することはできません。

ユーザーの権利

ユーザーは、本アプリが保有する自身の個人情報について、以下の請求を行うことができます。

  • 開示請求: 保有する個人情報の開示を請求する権利
  • 訂正・追加・削除請求: 内容が事実でない場合、訂正・追加・削除を請求する権利
  • 利用停止・消去・第三者提供停止請求: 法令に反する利用等があった場合に、利用停止・消去・第三者提供の停止を請求する権利

これらの請求は、上記のお問い合わせ窓口（tenderloin.games@gmail.com）までメールでご連絡ください。本人確認の上、法令に基づき、原則として30日以内に対応いたします。

なお、ご自身のアカウントと関連データの一括削除については、アプリ内の「アカウント削除とデータ消去」機能または専用のウェブページからも随時実施できます。

お子様のプライバシー

本アプリは18歳未満のお子様を対象としていません。18歳未満のお子様から意図的に個人情報を収集することはありません。

通報への対応

ユーザーから通報を受けた投稿については、運営者および当該歌会の主宰が原則として24時間以内に内容を確認し、利用規約に違反すると判断した場合は当該コンテンツの削除および違反したユーザーへの措置を速やかに講じます。詳細は利用規約の「通報への対応」の項をご参照ください。

本ポリシーの変更

本ポリシーは予告なく変更されることがあります。変更後のポリシーは本ページに掲載された時点で効力を生じます。
